# Instruções de utilização

## Instalação do Site
O site em HTML/CSS/JS é um projeto estático, logo pode ser utilizado tanto em servidores web quanto diretamente no navegador local.

### Opção 1: Execução Local (Recomendada para testes)
1. Faça o download ou clone o repositório do projeto;
2. Navegue até a pasta do projeto;
3. Abra o arquivo index.html diretamente no seu navegador preferido (Chrome, Firefox, Edge, etc).

### Opção 2: Usando um Servidor Local
Para uma experiência mais próxima do ambiente de produção, você pode usar um servidor local:

#### Usando a extensão Live Server (VS Code):
1. Instale a extensão "Live Server" no Visual Studio Code;
2. Clique com o botão direito no arquivo "index.html";
3. Selecione "Open with Live Server".

### Opção 3: Hospedagem em Servidor Web
O projeto pode ser hospedado em qualquer servidor web (Apache, Nginx) ou plataformas como:
- GitHub Pages;
- Reader;
- Netlify;
- Vercel.

Basta fazer upload dos arquivos para o servidor ou conectar o repositório à plataforma escolhida.
